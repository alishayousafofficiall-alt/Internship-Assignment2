Weather App Usage Guide

1. Features

  • Enter a city name to fetch weather details.
  • See temperature, humidity, and weather conditions.
  • Dynamic weather icons: Sunny, Rainy, Cloudy.
  • Dynamic background images change according to weather.

2. How to Use

  1. Open the app in your browser.
  2. Enter the city name in the input box.
  3. Click the "Get Weather" button.
  4. The weather details and icon will display instantly.
  5. The background image will update dynamically using Unsplash images.

3. Example Usage

Enter London and click "Get Weather" → the app will show temperature, humidity, and icon for London, and the background will update to a relevant image from Unsplash.

4. Notes

  • Ensure you have an internet connection to fetch weather and images.
  • City names should be in English for correct API results.
  • Background images are fetched using https://source.unsplash.com.

5. Troubleshooting

  • If the weather data doesn’t show, check your API key and network.
  • Background may not load if Unsplash is down; refresh to try a new image.
